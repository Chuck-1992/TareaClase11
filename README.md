# Tarea Clase 11

## Descripción

Este proyecto consiste en un programa desarrollado en Python para trabajar con arreglos unidimensionales, también conocidos como listas. El programa permite al usuario ingresar una cantidad mínima de cinco números y luego realiza diferentes operaciones utilizando bucles `for` y condiciones `if`.

Entre las operaciones principales se encuentran:

- Mostrar la lista original de números ingresados.
- Mostrar los números en orden inverso.
- Filtrar y mostrar los números mayores que 10.
- Calcular la suma total de todos los elementos.
- Contar la cantidad de números mayores que 10.
- Generar un reporte en formato PDF con los resultados obtenidos.
